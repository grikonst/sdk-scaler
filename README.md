Основной исполняемы файл: sdk-scaler\
Утилита для балансировки нагрузки(nginx) между несколькими инстасами luna-remote-sdk \
Конфигурационный файл: ~/.remote-sdk-manager.conf (создается автоматически) \
Конфиг nginx: ./nginx.conf (генерируется автоматически)

Установить пакеты:\
apt install -y dialog whiptail

Выполнить в терминале:\
#chmod +x sdk-scaler && sh sdk-scaler && Выбрать "1" меню и нажать "Enter"
